## Prerequisites

1. Latest version of Node to be installed.

## NPM Global packages
```
    npm install ts-node -g
    npm install typescript-node -g
    npm install babel-cli -g
```

## Typings Global Packages 
```
    typings install dt~mocha --global --save
```

## Setup
```
    npm install   
    node gensalt.js
```

## Test
```
    npm test
```
